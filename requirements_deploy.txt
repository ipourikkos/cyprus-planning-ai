# Cyprus Planning AI — Render Deployment

This folder is ready to deploy as one Render Web Service.

## Render settings

- Language: Python 3
- Build Command:
  `pip install -r requirements_deploy.txt`
- Start Command:
  `uvicorn planning_app:app --host 0.0.0.0 --port $PORT`
- Health Check Path:
  `/health`

## Environment variables to add in Render

Add these values in the Render dashboard. Do not commit them to GitHub.

- `SUPABASE_URL`
- `SUPABASE_SECRET_KEY`
- `OPENAI_API_KEY`

Optional:
- `OPENAI_CHAT_MODEL`
- `OPENAI_QUERY_EXPANSION_MODEL`
- `OPENAI_VERIFY_MODEL`

## What is included

- `planning_app.py` — serves both the API and frontend
- `planning_api.py` — FastAPI API
- `ask_planning_ai_v11_final_backend.py` — tested planning AI engine
- `web/` — browser interface
- `.python-version` — Python runtime version
- `requirements_deploy.txt` — deployment dependencies

The `.env` file is intentionally NOT included.
